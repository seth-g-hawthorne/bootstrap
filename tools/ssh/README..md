# Launchpad: tools/ssh

## Commands
  * `eval $(ssh-agent) && ssh-add` - restart ssh agent and add keys from ~/.ssh/id_rsa
  * `ssh-keyscan <hostname> >> ~/.ssh/known_hosts` - Pre-load ssh keys
