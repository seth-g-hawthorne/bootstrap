# Launchpad: k8s/K3d
K3d is a wrapper script that makes it easy to run K3s clusters.

## Usage
   1. `bin/k3d-install` - Install K3d and dependencies
   1. Use the `k3d` command to create and manage clusters; enter `k3d` for help
